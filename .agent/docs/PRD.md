# 프로젝트명: 재무컨설팅 전문 기업 웹사이트

## 1. 목적 및 비전
- 보험 및 재무 설계 전문가로서의 신뢰감을 주는 웹사이트 구축
- 교보생명(wholelife.co.kr)의 깔끔하고 정돈된 톤앤매너 벤치마킹
- 잠재 고객의 상담 문의 유도 (Lead Generation)

## 2. 타겟 사용자
- 개인 재무 설계 및 보험 리모델링이 필요한 일반 고객
- 전문적인 기업 인증 컨설팅을 원하는 기업 관계자

## 3. 핵심 기능 (MVP)
- **홈 (Hero Section):** 신뢰감 있는 이미지와 핵심 슬로건 배치 (고객의 페인포인트 공감)
- **서비스 소개:** - 개인 재무 설계 (보험, 은퇴, 자산 관리)
  - 기업 인증/금융 컨설팅 (보험 대리점 및 금융 컨설팅 역량 강조)
- **전문가 프로필:** 컨설턴트의 이력과 전문성 강조 (신뢰도 향상)
- **상담 신청 (CTA):** 간편 상담 신청 폼 (이름, 연락처, 관심 분야)
- **블로그/공지사항:** 재무 상식 및 기업 소식 게시판

## 4. 디자인 가이드라인 (Benchmark: 교보생명)
- **컬러:** 신뢰를 상징하는 딥 블루(Deep Blue) 또는 화이트/그레이 조합
- **레이아웃:** 넓은 여백(White Space)을 활용한 미니멀한 구성
- **요소:** 둥근 모서리의 카드 UI, 고해상도 인물 사진, 가독성 높은 산세리프 폰트

## 5. 기술 스택 (권장)
- Framework: Next.js 14+ (SEO 최적화 및 빠른 로딩)
- Styling: Tailwind CSS (일관된 디자인 시스템)
- UI Library: shadcn/ui (교보생명 스타일의 깔끔한 컴포넌트 구현)
